# dotfiles

My dotfiles setup managed by chezmoi.

## Description

My dotfiles configs for Neovim, Fish and Zellij.

## Getting Started

### Installation

```
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply dalvnx
```

### Tools and applications

* Fish (shell)
* Neovim (code editor)
* Zellij (windows)
